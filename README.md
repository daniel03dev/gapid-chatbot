# Sistema Conversacional Inteligente para GAPID

Sistema de chatbot conversacional para apoyo a usuarios del Sistema de Programas y Proyectos de Ciencia, Tecnología e Innovación (GAPID) de Cuba.

## 🎯 Objetivo

Facilitar el acceso y comprensión de normativas, procedimientos y guías relacionadas con la gestión de proyectos de CTI mediante un asistente conversacional inteligente basado en Recuperación de Información Aumentada (RAG).

## 📚 Documentos Fuente

El sistema responderá consultas basándose en:
1. **GOC-2025-O13**: Reglamento del Sistema de Programas y Proyectos
2. **Guía TRL**: Niveles de Madurez Tecnológica
3. **Manual SPP**: Manual de Gestión del Sistema

## 🏗️ Arquitectura (Planificada)

- **Backend**: Django + Django REST Framework
- **Frontend**: Next.js con TypeScript
- **Base de Datos**: PostgreSQL
- **Vectorización**: FAISS + sentence-transformers
- **Contenedorización**: Docker + Docker Compose
