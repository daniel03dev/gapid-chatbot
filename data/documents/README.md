# Documentos Fuente

Coloca aquí los documentos en formato `.txt` que serán procesados por el sistema:

- `GOC-2025-O13.txt` - Reglamento del Sistema de Programas y Proyectos
- `Guia_TRL.txt` - Guía de Niveles de Madurez Tecnológica
- `Manual_SPP.txt` - Manual de Gestión del Sistema

Los documentos deben estar en formato texto plano (UTF-8) para ser procesados correctamente por el sistema de vectorización.
