MODELOS DE DATOS
-----------------
✅ Conversation
   - ID autogenerado
   - Timestamps de creación y actualización
   - Relación con mensajes

✅ Message
   - Roles: user/assistant
   - Contenido del mensaje
   - Relación con conversación
   - Timestamps

✅ QueryLog (NUEVO - 28/02/2026)
   - Registro de cada consulta del usuario
   - Respuesta del asistente
   - Tiempo de respuesta en segundos
   - Cantidad de chunks recuperados
   - Contexto completo usado del índice vectorial
   - IP del cliente
   - User-agent del navegador
   - Timestamp de la consulta
   - Campo para feedback opcional (1-5)

✅ AuditLog (NUEVO - 28/02/2026)
   - Eventos del sistema (query, index_build, error, config_change, startup)
   - Niveles de severidad (info, warning, error, critical)
   - Descripción del evento
   - Metadata adicional en JSON
   - IP y user-agent
   - Timestamp del evento

SERVICIOS RAG (Retrieval-Augmented Generation)
-----------------------------------------------
✅ DocumentProcessor
   - Procesamiento de archivos .txt
   - Chunking de documentos (500 caracteres, overlap 100)
   - Extracción de metadata (nombre archivo, chunk ID)

✅ VectorizerService
   - Modelo: paraphrase-multilingual-MiniLM-L12-v2
   - Embeddings de 384 dimensiones
   - Índice FAISS para búsqueda vectorial
   - Optimizado para CPU
   - Persistencia en disco (faiss_index.bin, chunks.pkl)

✅ ChatService
   - Construcción de índice vectorial
   - Búsqueda de contexto relevante (top-k)
   - Generación de respuestas basadas en documentos
   - Sistema de logging automático (NUEVO)
   - Registro de eventos de auditoría (NUEVO)
   - Cálculo de tiempo de respuesta (NUEVO)
   - Extracción de IP y user-agent (NUEVO)

ENDPOINTS API
-------------
✅ GET /api/status/
   - Health check del sistema

✅ POST /api/chat/
   - Procesar consulta del usuario
   - Crear conversación si no existe
   - Guardar mensajes (user + assistant)
   - Retornar respuesta con fuentes
   - LOGGING AUTOMÁTICO ACTIVADO (NUEVO)
   - Incluye response_time y chunks_retrieved (NUEVO)

✅ GET /api/conversations/
   - Listar todas las conversaciones
   - Incluye contador de mensajes

✅ POST /api/conversations/
   - Crear nueva conversación

✅ GET /api/conversations/<id>/
   - Detalle de conversación con todos sus mensajes

✅ GET /api/conversations/<id>/messages/
   - Listar mensajes de una conversación

✅ POST /api/conversations/<id>/messages/
   - Crear nuevo mensaje en una conversación

✅ GET /api/logs/queries/ (NUEVO - 28/02/2026)
   - Listar consultas registradas
   - Filtros: limit, conversation_id
   - Preview de consultas y respuestas

✅ GET /api/logs/queries/<id>/ (NUEVO - 28/02/2026)
   - Detalle completo de una consulta
   - Incluye contexto completo usado

✅ GET /api/logs/audit/ (NUEVO - 28/02/2026)
   - Listar eventos de auditoría
   - Filtros: limit, event_type, severity

✅ GET /api/metrics/ (NUEVO - 28/02/2026)
   - Total de consultas y conversaciones
   - Tiempo promedio de respuesta
   - Chunks promedio recuperados
   - Consultas en últimas 24h, 7d, 30d
   - Promedio de feedback (si existe)
   - Total de errores
   - Horas más activas del sistema
